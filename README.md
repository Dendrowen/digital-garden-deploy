# digital-garden-deploy
Script to deploy digital garden on a debian machine

# Prerequisites
- install libatomic `apt install libatomic`
- install npm through nvm:
	- `curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.5/install.sh | bash`
	- nvm install 24
	- nvm use 24

# Usage
`curl -o- https://raw.githubusercontent.com/Dendrowen/digital-garden-deploy/refs/heads/main/script.sh | bash`
